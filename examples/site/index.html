patinate

Strava activity rendered as a stylized heatmap poster. The hero below is the bundled fixtures/activities.json (30 synthetic rides centered on Grand Rapids) rendered into an inline-embeddable SVG via patinate render --web. Filters demonstrate the data-* attributes the renderer emits on every heat path.

What you're looking at

The basemap is OSM road geometry (water lines + line-water + parks + road tiers). The heat layer is one path per activity, drawn with multiply blend mode so overlapping rides accumulate into brighter color. The patina builds where you ride often.

Embed it yourself

The full README at the patinate repo walks through install, OAuth, sync, and render. The web preset targets ~500KB SVG with single-layer heat and 1-digit coord precision; pair with --transparent-bg to inline on a page that already provides its own background.

Built with patinate. Source at gitlab.com/dunn.dev/patinate.
